1. Open `templates.html` in your code editor
2. Locate the template section (commented with `<!-- TEMPLATE FOR NEW PRODUCTS -->`)
3. Copy the template structure
4. Paste it below the last product card in the `templates-grid` div
5. Replace the placeholder values:
   - `CATEGORY_HERE`: Choose from content, automation, chatbot, nologin, pdf
   - `PRICE_HERE`: Set the product price (e.g., 49.99)
   - `YYYY-MM-DD`: Set the current date for sorting purposes
   - `IMAGE_NAME.png`: Use an image from the Phtots directory
   - `PRODUCT_NAME`: Set your product name
   - `PRODUCT_DESCRIPTION`: Add a compelling description
   - `TECH_1`, `TECH_2`, `TECH_3`: List technologies used
   - `DETAILS_PAGE_URL`: Link to product details (e.g., bundle.html)
   - `LIVE_PREVIEW_URL`: Link to live demo
   - `PURCHASE_URL`: Link to purchase page
6. Save the file and upload to your hosting
